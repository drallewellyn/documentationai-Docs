<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Example: Integrate AI tool via simple API call
  const response = await fetch('https://api.example.com/ai/career-advice', {
    method: 'POST',
    headers: { 'Authorization': 'Bearer YOUR_API_KEY' },
    body: JSON.stringify({ specialty: 'general-practice', years: 5 })
  });
  const advice = await response.json();
  console.log(advice.recommendations);
  ```
  ```python
  import requests
  response = requests.post(
      'https://api.example.com/ai/career-advice',
      headers={'Authorization': 'Bearer YOUR_API_KEY'},
      json={'specialty': 'general-practice', 'years': 5}
  )
  advice = response.json()
  print(advice['recommendations'])
  ```
</CodeGroup>
